# Bitcoin_Price
 the bitcoin price envolving mercadobitcoin.com API in shellscript
## how to use (shellcript)
* git clone https://github.com/RaelzeraXD/bitcoin_price
* cd bitcoin_price
* ./btcprice.sh

## how to use (python)
* sudo apt install python3 python3-requests -y
* git clone https://github.com/RaelzeraXD/bitcoin_price
* cd bitcoin_price
* ./btcprice.py
